# AL OMRANE BI Project

Projet Business Intelligence pour la société immobilière AL OMRANE.

L'objectif actuel est de préparer l'environnement technique, créer une base transactionnelle PostgreSQL, générer un dataset avec Python/Faker, puis l'utiliser avec Apache Hop pour construire un Data Warehouse.

---

# Stack Technique

- Linux
- Docker
- PostgreSQL 16
- pgAdmin
- Python
- Faker
- Pandas
- Apache Hop
- VS Code

# Installation Apache Hop

Apache Hop remplace Pentaho Data Integration (PDI) comme orchestrateur ETL du projet.

## Pourquoi Apache Hop ?

Hop est le successeur open-source de Pentaho. Il offre :
- Même paradigme ETL (pipelines, workflows)
- Interface graphique Hop GUI (ex-Spoon)
- Meilleure gestion des métadonnées
- Actif et maintenu

Les pipelines et workflows Hop sont versionnés directement dans le dépôt (dossier `hop/`).

---

## Prérequis

Installer Java 17 :

```bash
sudo apt update
sudo apt install openjdk-17-jdk unzip
```

Vérification :

```bash
java --version
```

---

## Installation

Télécharger Apache Hop depuis :

```text
https://hop.apache.org/download/
```

Extraire l'archive :

```bash
mkdir -p ~/tools
cd ~/tools
unzip apache-hop-*.zip
```

La structure obtenue :

```text
~/tools/
└── apache-hop/
    ├── hop-gui.sh
    ├── hop-conf.sh
    ├── lib/
    └── ...
```

---

## Lancement de Hop GUI

```bash
cd ~/tools/apache-hop
chmod +x hop-gui.sh
./hop-gui.sh
```

---

## Driver PostgreSQL

Télécharger le driver JDBC PostgreSQL :

```text
postgresql-42.x.x.jar
```

Copie du driver :

```bash
cp ~/Downloads/postgresql-42*.jar \
~/tools/apache-hop/lib/
```

Relancer ensuite Hop GUI.
